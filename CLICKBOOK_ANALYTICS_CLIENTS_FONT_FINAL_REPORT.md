# ClickBook final analytics / clients / font repair

Изменения:
- глобальный стек шрифта переключён на Golos UI / Golos Text / system-ui без встраивания font-файлов в архив;
- аналитика теперь считает заявки и посетителей по дате создания заявки, а не по будущей дате визита;
- будущий createdAt из старых данных зажимается текущим днём, чтобы 2 мая не появлялось как факт, когда сегодня 1 мая;
- дневная аналитика строится за последние 30 дней до текущего дня, без будущих дат;
- недельный график на главной строится по текущей календарной неделе ПН–ВС;
- подробная аналитика теперь отличается от активности по дням: она показывает разрезы по услугам/каналам, а не копию дневного графика;
- клиент после статуса no_show / cancelled переводится в сегмент «Спящий»;
- добавлена ручная VIP-пометка клиента через CRM-карточку;
- VIP/заметки на главной стали кликабельными и открывают мини-попап;
- добавлена безопасная SQL-миграция 0015 для нормализации MAX -> ВК и new -> confirmed.

Важно по шрифту: font-файлы не включены в архив. Если нужен именно локальный Golos UI, скопируй свои лицензированные .ttf в public/fonts/golos-ui и добавь @font-face. Сейчас CSS уже использует font-family: Golos UI, Golos Text, system-ui.
